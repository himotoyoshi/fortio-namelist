### グループ名のプレフィックス

'&'と'$'が使える。グループの終端を表す'end'のプレフィックスも同様であるが、
始端と終端のプレフィックスのペアのチェックはしない。

### 文字列の配列
 
    v1 = "a", "b", 'c'  ! 引用符あり文字列の配列
    v2 = a, b, c        ! 引用符なし文字列の配列
    v3 = a, 0_b, _c     ! 引用符なし文字列の配列

ただし、引用符あり文字列と引用符なし文字列を混在させることはできない。

    v4 = "a", b, "c"    ! エラー 
    v5 = "a", 0_b, c    ! エラー

### 論理型リテラル

    v1 = .true.         ! true
    v2 = .t             ! true
    v3 = .t_1           ! true
    v4 = t              ! true
    
    v4 = .false.        ! false
    v5 = .f             ! false
    v5 = .f_1           ! false
    v6 = f              ! false

もちろん大文字でもよい。
